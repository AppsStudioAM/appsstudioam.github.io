# Apps Studios AM Website

Professional gaming portfolio website showcasing 12 Android games.

## Hosting (Free)
1. **GitHub Pages** - Upload to GitHub, Settings > Pages > Deploy
2. **Netlify** - Drag and drop at netlify.com
3. **Vercel** - Import from GitHub at vercel.com

## Customize
- Edit `js/games-data.js` to modify games
- Edit `css/style.css` to change styling
- Add Play Store URLs in game data
